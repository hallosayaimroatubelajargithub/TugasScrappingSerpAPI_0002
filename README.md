# TugasScrappingSerpAPI_0002
Melakukan scrapping data lokasi restoran KCF yang ada di kota Semarang menggunakan SerpAPI. 
Dalam tugas tersebut terdapat beberapa Preprocessing, yaitu :
1. Case folding
2. Tokenizing
3. Stopword removal
4. Stemming
